## Have `^=` and `$=` match sizzle's behavior for empty strings

```javascript
console.log( $('<div id="myDiv" test=""></div>').is("[test^='']") );
console.log( $('<div id="myDiv" test=""></div>').is("[test$='']") );
console.log( $('<div id="myDiv" test=""></div>').is("[test^=]") );
console.log( $('<div id="myDiv" test=""></div>').is("[test$=]") );
```

All of them return **false** for sizzle. We don't right now.

This should have very low priority.
